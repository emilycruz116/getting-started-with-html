Unit 1

The World Wide Web is the collection of HTML documents.
HTML stands for Hyper Text Markup Launguage
HTTP is the main protocol of the web.
Servers are computers that host the files that makeup the web.
The Internet is the the worlds largest computer network.
A browser is a program that runs on your computerr to display files found on the web.

HTML is the heart of the web.
HTML is made up of text content, markup, references to other documents (images and videos),and links to other pages.
HTML markup consist elements, and the different types elements are distinguished by tag.

There are two types of HTML elements: Inline and Block

Inline elements can be a, span, br, img, and strong.
Block elements can be div, p and form.
